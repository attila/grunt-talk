If you don't like Grunt

Grunt

  • gruntjs.com

  • One of the first around the block

  • Vast number of plugins

  • extensible API → gruntjs.com/api/grunt

  • scaffolder tool to build plugins

  • separate, initially independent steps begin to depend on each other

Gulp

  • gulpjs.com

  • Preferred over grunt by many

  • Uses pipes

  • Separate plugins are not needed most of the time

Metalsmith

  • metalsmith.io

  • Labelled as a static site generator

  • Can be used as a build tool, project scaffolder, documentation generator, etc

  • "Everything is a plugin" approach, chain them all together

Makefile

  • www.gnu.org/software/make/manual/make.html

  • Ubiquitous

  • Its syntax is a bit obscure ☛

  • Very hip
